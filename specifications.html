Deliverables

Deliverables for this group project will include:

  • A front end, multiple-page website utilizing React.js and A-Frame web framework
  • Functionality enabling users to personalize and manipulate a 3D object of a product
  • A backend, using Node.js for storing and retrieving data such as user login credentials
  • Functionality enabling users to log in, access and modify personalized data
  • A brand guide, specifying brand mission, vision, typography, color use, logo use, and content organization.

Database Design

The database for this project will include data such as login credentials, a user ID, wishlist items, and product customizations. The primary key will be the user ID.
